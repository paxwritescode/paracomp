# Параллельные методы решения задачи Коши для системы ОДУ

## Исследования

1. Зависимость времени выполнения от размера системы
2. Зависимость времени выполнения от числа параллельных процессов/потоков
3. Зависимость ускорения от числа параллельных процессов/потоков
4. Зависимость эффективности параллелизации от числа параллельных процессов/потоков

## Варианты

1. Метод Пикара
2. Метод Рунге-Кутты 4 порядка (явный)
3. Метод Адамса предиктор-корректор
